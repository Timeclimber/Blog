# 本地个人博客

这是一个使用Go语言开发的本地个人博客系统，具有以下功能：

- 文章管理：创建、查看文章
- 评论系统：为文章添加评论
- 标签系统：为文章添加标签
- 数据持久化：使用JSON文件存储数据

## 技术栈

- 后端：Go标准库
- 前端：HTML、CSS、JavaScript
- 数据存储：JSON文件

## 项目结构

```
本地个人博客/
├── web/
│   ├── templates/          # 前端模板
│   │   ├── index.html      # 首页
│   │   ├── new.html        # 写文章页面
│   │   └── article.html    # 文章详情页面
│   └── static/             # 静态文件
│       └── css/            # CSS样式
│           └── style.css   # 主样式文件
├── simple_blog.go          # 主程序文件
├── blog_data.json          # 数据存储文件
└── README.md               # 项目说明文档
```

## 运行方法

1. 确保你已经安装了Go语言环境
2. 进入项目目录
3. 运行以下命令启动服务器：

```bash
go run simple_blog.go
```

4. 打开浏览器访问：http://localhost:8081

## 功能说明

### 首页
- 显示所有文章列表
- 点击文章标题进入文章详情页面

### 写文章
- 填写文章标题和内容
- 可以添加标签（用逗号分隔）
- 点击"发布文章"按钮提交

### 文章详情
- 显示文章内容和标签
- 显示文章的评论
- 可以添加新评论

## 数据存储

系统使用`blog_data.json`文件存储所有数据，包括：
- 文章信息
- 评论信息
- 标签信息
- 文章与标签的关联信息

## 注意事项

- 这是一个本地博客系统，数据存储在本地文件中
- 每次启动服务器时会加载已有的数据
- 每次添加或修改数据时会自动保存到文件中
- 服务器运行在8081端口，确保该端口未被占用
